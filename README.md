# bigEvent项目文档

## 1.介绍

* 文件介绍

  * web_back和web_front是静态页面
  * bigEventServer是本地服务器，内部配有接口文档

* 项目介绍：

  * 大事件是一个具有前后台的**内容管理网站**
  * 前台
    * 是用来给用户查看的页面，只能用于**数据的展示** 
  * 后台
    * 是用来给网站管理员使用的页面，可以进行网站数据管理（增删改查）


* web_back
* web_front
    * 前端
      * 是采用htmlcssjs进行开发的
    * 后端
      * 是采用java\php\python\nodejs
